# Goto foo
Adds a 'Goto &lt;url&gt;' option to Firefox, when right-clicking text that resembles a URL.

Note that Chrome already has a similar feature built in.

## Add to Firefox
https://addons.mozilla.org/firefox/addon/gotofoo/
